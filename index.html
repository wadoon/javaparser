JML Parser¶

This project provides a library for the parsing of Java with the Java Modelling Language (JML). JML is a formal specification for Java to decribe the functional behavior, e.g., pre- and post-conditions of methods, class and loop invariants.

The bases of this project is the [Java Parser]https://github.com/javaparser/javaparser) project, which is extended in the following ways:

  • new lexer and grammar rules
  • new AST classes and also new attributes for Block, WhileStatement, ForStatement, DoStatement, and CallableDeclaration (constructors and methods).
  • (TODO) extension of the name resolution
